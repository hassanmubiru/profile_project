🛠️ TROUBLESHOOTING DEPLOYMENT
════════════════════════════════════════════════════════════════════════════

Problem: "Python not found"
Solution: Install Python 3.8+ from https://www.python.org/

Problem: "venv: No module named 'venv'"
Solution: Install python3-venv
  Ubuntu/Debian: sudo apt install python3-venv
  macOS: brew install python3
  Windows: Reinstall Python with venv checked

Problem: "pip not found"
Solution: Use: python -m pip install Django==4.2.0

Problem: "Permission denied" on DEPLOY.sh
Solution: chmod +x DEPLOY.sh

Problem: Port 8000 in use
Solution: python manage.py runserver 8001

Problem: Database errors
Solution: Delete db.sqlite3 and run: python manage.py migrate
