⏱️ 5-MINUTE DEPLOYMENT FLOW
════════════════════════════════════════════════════════════════════════════

MINUTE 1: Setup Git & GitHub
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Open terminal in project folder and run:

  git init
  git add .
  git commit -m "Django Profile Project"


Then on GitHub.com:
  1. Click "+" → "New repository"
  2. Name it "profile-project"
  3. Click "Create repository"
  4. Copy the commands shown and run them in terminal:

    git remote add origin https://github.com/YOUR_USERNAME/profile-project.git
    git branch -M main
    git push -u origin main

Done! Your code is on GitHub.


MINUTE 2-3: Render Setup
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Go to https://render.com:
  1. Sign up (GitHub login easiest)
  2. Click "New +" → "Web Service"
  3. Connect your GitHub repository
  4. Select "profile-project" repo
  

Fill in form:
  Service Name: profile-project
  Environment: Python 3.11
  Plan: Free

That's it! Click "Create Web Service" and wait...


MINUTE 4-5: Wait & Verify
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Render builds and deploys automatically:
  • Takes 3-5 minutes
  • Shows "Your service is live!" when done
  • Copy your URL: https://profile-project-xxx.onrender.com


🎉 YOU'RE LIVE!

Your app is on the internet!
  • Test it: https://profile-project-xxx.onrender.com/admin/
  • Share the URL: https://profile-project-xxx.onrender.com/profile/


⚙️ CREATE SUPERUSER (First Admin Access)
════════════════════════════════════════════════════════════════════════════

After deployment, in Render Dashboard:
  1. Click on your service (profile-project)
  2. Click "Shell" tab
  3. Run: python manage.py createsuperuser
  4. Create admin user

Then login: https://profile-project-xxx.onrender.com/admin/


📝 ADD YOUR PROFILE DATA
════════════════════════════════════════════════════════════════════════════

After login to admin:
  1. Click "Profiles" → "Add Profile"
  2. Fill in:
     - User: admin
     - Age: 25
     - Bio: My Django App
     - Topics: variables, loops, functions
  3. Save!
  4. Visit: https://profile-project-xxx.onrender.com/profile/


🔄 UPDATE YOUR SITE (AUTOMATIC!)
════════════════════════════════════════════════════════════════════════════

Whenever you update code:
  1. Make changes locally
  2. Run in terminal:

    git add .
    git commit -m "Your changes"
    git push

  3. Render automatically rebuilds and deploys!
  4. Site updates in 2-3 minutes


❓ SOMETHING WRONG?
════════════════════════════════════════════════════════════════════════════

Check Render Dashboard → Your Service → Logs

Common issues:
  • "Build failed" → Check Logs, usually missing dependencies
  • "Cannot connect" → Check Logs for startup errors
  • "Page not found" → Make sure to visit /admin/ or /profile/
