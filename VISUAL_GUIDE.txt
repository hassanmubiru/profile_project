📊 DATA FLOW VISUALIZATION
═══════════════════════════════════════════════════════════════════════════════

User visits /profile/
    ↓
Django checks: @login_required (is user logged in?)
    ↓
View creates VARIABLES:
    • current_user = logged-in user object
    • today_topics = ["variables", "loops", "functions"]
    • user_data = {username: ..., email: ...}
    ↓
View creates a LOOP:
    for index, topic in enumerate(today_topics):
        Create formatted_topic strings
    ↓
View returns CONTEXT (variables to template):
    context = {
        "user_data": user_data,
        "today_topics": today_topics,
        ...other variables...
    }
    ↓
Template displays VARIABLES:
    {{ user_data.username }}
    {{ profile.age }}
    ↓
Template runs LOOP:
    {% for topic in today_topics %}
        <li>{{ forloop.counter }}: {{ topic }}</li>
    {% endfor %}
    ↓
Beautiful HTML is rendered
    ↓
User sees profile page! ✓
