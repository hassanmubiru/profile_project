🔄 LEARNING CONTENT - LOOPS
═══════════════════════════════════════════════════════════════════════════════

PYTHON FOR-LOOP EXAMPLE (views.py, lines 57-68):

today_topics = ["variables", "loops", "functions"]
processed_topics = []

for index, topic in enumerate(today_topics, start=1):
    # Loop body executes 3 times (once per item)
    # index: 1, 2, 3 (position indicator)
    # topic: "variables", "loops", "functions" (current item)
    
    formatted_topic = f"Lesson {index}: {topic.upper()}"
    processed_topics.append(formatted_topic)

RESULT:
["Lesson 1: VARIABLES", "Lesson 2: LOOPS", "Lesson 3: FUNCTIONS"]


DJANGO TEMPLATE FOR-LOOP (profile.html, lines 127-145):

{% for topic in today_topics %}
    <li class="topic-item">
        <span class="topic-number">{{ forloop.counter }}</span>
        <span class="topic-name">{{ topic|upper }}</span>
    </li>
{% endfor %}

Template variables:
• forloop.counter    - Current iteration (1, 2, 3...)
• topic              - Current item from list
• forloop.first      - True on first iteration
• forloop.last       - True on last iteration
• forloop.length     - Total number of items


🚀 SETUP INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════════

QUICK START (3 steps):

1. Install Django:
   pip install Django==4.2.0

2. Setup database:
   python manage.py migrate
   python manage.py createsuperuser

3. Run server:
   python manage.py runserver

Then visit:
• Admin:   http://localhost:8000/admin/
• Profile: http://localhost:8000/profile/


DETAILED SETUP WITH VIRTUAL ENVIRONMENT:

# Navigate to project
cd profile_project

# Create virtual environment
python3 -m venv venv

# Activate (Linux/Mac):
source venv/bin/activate
# Or (Windows):
# venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Setup database
python manage.py migrate

# Create admin account
python manage.py createsuperuser
# Follow prompts, example:
# Username: admin
# Email: admin@example.com
# Password: password123

# Start server
python manage.py runserver
