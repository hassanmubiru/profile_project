>>> TROUBLESHOOTING <<<

❌ "Module not found" error?
   → Activate virtual environment: source venv/bin/activate
   → Install dependencies: pip install -r requirements.txt

❌ "Table doesn't exist" error?
   → Run migrations: python manage.py migrate

❌ Can't login to admin?
   → Create superuser: python manage.py createsuperuser

❌ /profile page shows login page?
   → Login via /admin/ first
   → Then visit /profile/

>>> EDIT PROFILE DATA <<<

Method 1: Django Admin Panel
  1. Visit http://localhost:8000/admin/
  2. Click "Profiles"
  3. Click on a profile to edit
  4. Modify: age, bio, topics (as JSON list)
  5. Click "Save"

Method 2: Django Shell
  python manage.py shell
  
  from profile_app.models import Profile
  from django.contrib.auth.models import User
  
  user = User.objects.first()
  profile = Profile.objects.get(user=user)
  profile.age = 25
  profile.bio = "Learning Django"
  profile.topics = ["variables", "loops", "functions"]
  profile.save()
