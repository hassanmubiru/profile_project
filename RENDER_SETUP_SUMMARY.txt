📞 NEED HELP?
════════════════════════════════════════════════════════════════════════════

Issue: "Build failed on Render"
→ Check Render Logs tab
→ Usually: missing dependency in requirements.txt

Issue: "Application crashed"
→ Render Dashboard → Logs tab
→ Shows exact error

Issue: "Page not loading"
→ Check URL is: https://profile-project-xxx.onrender.com
→ Not: http:// (must be https)

Issue: "404 Not Found"
→ Try: /admin/ or /profile/
→ Check URLs in profile_project/urls.py
