🚀 QUICK START (3 Commands)
═══════════════════════════════════════════════════════════════════════════════

1. Install Django:
   pip install Django==4.2.0

2. Setup database:
   cd profile_project
   python manage.py migrate
   python manage.py createsuperuser

3. Start server:
   python manage.py runserver

Then visit:
   http://localhost:8000/admin/     (login with superuser account)
   http://localhost:8000/profile/   (view your profile with variables & loops)
